# 🌐 Sri Portfolio

A modern and responsive personal portfolio website built using **React.js** and **Vite** to showcase my skills, projects, resume, and contact information.

## 🚀 Live Demo


---https://sudisrinadhcom.netlify.app/

## 📌 Features

- Responsive design for desktop and mobile
- Modern and clean user interface
- Hero section with profile information
- About Me section
- Skills section
- Projects showcase
- Resume download option
- Contact section
- Smooth navigation and animations

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/sri_portfolio.git
```

Go into the project directory

```bash
cd sri_portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
